# computing Devices and Hardware components
## What is a computing device?
A computing device is an electronic device that process data according to the set of instruction
### Example:
Laptop, Desktop, Tablet, Servers.

## Important Components
- CPU
- RAM
- Storage
- Network Interface Card(NICs)
- GPU
# Software and Operating System

## Software:
Instruction that tell the computing device what to do 
## Types of Software:
- System Software
- Application Software
- Programming Software

## Diffrent types of Application Used:
- Desktop Applications
- Mobile Applications
- Web Applications
- Cloud Applications
